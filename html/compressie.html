Compressie

Het klinkt onmogelijk, maar je kan de grootte
van bestanden verkleinen. Dit kan d.m.v. compressie.

Compressie bij afbeeldingen

Er zijn twee manieren van compressie: lossy en
lossless.

Bij lossy compressie worden delen van een bestand verkleind of
verwijdert. Dit gebeurt bijvoorbeeld bij afbeeldingen van hele hoge kwaliteit.
Als je een afbeelding hebt met heel veel verschillende kleuren in bijvoorbeeld
het haar, kan d.m.v. lossy compressie veel van deze kleuren weggehaald worden
waardoor de grootte van het bestand een stuk kleiner wordt en de kwaliteit niet
zichtbaar verandert, omdat het gaat om zulke kleine details die veel ruinmte innemen.

Bij lossless compressie wordt er niets verwijdert, de bestanden worden
alleen slimmer opgeslagen. Zo kan een computer een factor plaatsen van hoe vaak een bepaalde
code voorkomt, i.p.v. telkens de code opnieuw te lezen. Dit scheelt tijd!

Compressie van geluidsfragmenten

Bij geluidsfragmenten bespaard compressie ook opsluigruimte door
overbodige delen van het bestand te verwijderen.
Geluid wat niet te horen is door andere overlappende geluiden worden
bijvoorbeeld weggehaald.
